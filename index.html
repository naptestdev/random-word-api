Random Word API

GET /word

Get one random word

Parameters

Name Needed Usage
min Optional Minimum length of the word
max Optional Maximum length of the word

GET /words?num=10

Get list of random words

Parameters

Name Needed Usage
num Required Numbers of words to receive
min Optional Minimum length of the words
max Optional Maximum length of the words

GET /all?dl=1

Get all words on server's database

Parameters

Name Needed Usage
dl Optional Download the json or display on the browser
